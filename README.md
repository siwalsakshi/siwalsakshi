## 🌿 Who I Am

> *"Turning coffee, curiosity, and a lot of `print()` statements into working AI."* ☕🐍

I'm a **Computer Engineering student** at ABES Engineering College, currently trading textbook theory for real production code as a **Software Trainee at Anvya AI** — where I build AI-powered automation workflows, wire up RAG pipelines, and get LLMs to actually talk to APIs nicely.

My playground is **Python** — from teaching a webcam to referee Rock-Paper-Scissor with OpenCV, to building a movie recommender that argues with itself using TF-IDF and cosine similarity. When I'm not building things, I'm probably grinding out DSA problems — **1000+ solved on CodeChef** *(Diamond badge 💎)* and **215+ on LeetCode** — because apparently I like solving puzzles even outside of work.
